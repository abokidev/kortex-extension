# Kortex Browser Extension

Surface codebase memory anywhere you work — GitHub, Linear, Jira, Notion, Word Online, Google Docs.

## What it does

- Injects a Kortex sidebar on supported pages showing relevant memory from your codebase
- Auto-detects page context — GitHub PRs, Linear tickets, Jira issues, Notion pages, Word Online documents
- Watches live typing in document editors — type a symbol name and Kortex surfaces related decisions instantly
- Ask Kortex questions directly from the sidebar using natural language Q&A
- Affirm or deny proposed memories without leaving the current page
- Works for developers and non-technical team members alike — no CLI required

## Installation

### Chrome
1. Download or clone this repository
2. Open Chrome and go to chrome://extensions
3. Enable Developer mode (top right)
4. Click Load unpacked and select the kortex-extension folder
5. The Kortex icon appears in your toolbar

### Firefox
Coming soon via Firefox Add-ons.

## Setup

1. Click the Kortex icon in your toolbar
2. Sign in with your Kodingo account
3. Click Manage projects and add your project tokens from kodingo.xyz/dashboard
4. Visit any supported page — the Kortex sidebar appears automatically

## Keyboard shortcut

Cmd+Shift+K (Mac) or Ctrl+Shift+K (Windows/Linux) toggles the sidebar on any page.

## Supported surfaces

- GitHub pull requests and issues
- Linear issues
- Jira issues
- Notion pages
- Word Online documents
- Google Docs
